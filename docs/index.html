Tóm tắt & Phân tích AI

Tab Summary gom nhóm ý và giữ nguyên cách diễn đạt; tab Analyze phân tích tổng quan cùng câu hỏi/rủi ro. Mặc định chạy trên Qwen 2.5-7B local (Ollama) — riêng tư, offline; đổi sang Claude, Codex hay Gemini trong Settings.

100% cục bộ và riêng tư

Âm thanh không bao giờ rời khỏi máy. Nghe, chuyển chữ, dịch và tách người đều chạy offline — và với Qwen local mặc định thì cả Tóm tắt/Phân tích AI cũng 100% trên máy. Chỉ khi bạn tự đổi sang model cloud thì phần văn bản mới được gửi đi. Đầu ra mặc định là bản .md, tuỳ chọn kèm .txt, .mp3 và bản tóm tắt AI.

Cài đặt

Chạy được sau vài dòng lệnh

Clone repo, chạy script cài một lần để dựng môi trường, rồi khởi động. Lần đầu sẽ tải model về (khoảng vài GB), sau đó dùng hoàn toàn offline. Bước setup-ollama.ps1 là tuỳ chọn — cài Qwen local cho tab Summary/Analyze (mặc định); bỏ qua nếu chỉ dùng provider cloud.

  • Windows 11 và Python 3.12
  • Khuyến nghị GPU NVIDIA (CUDA) để transcribe realtime mượt
  • Script tự dựng venv, cài torch CUDA và tải model
Đọc hướng dẫn chi tiết
PowerShell 
git clone https://github.com/ducnguyen221/ai-record
cd ai-record
powershell -ExecutionPolicy Bypass -File setup.ps1              # cài một lần: venv + torch CUDA + model
powershell -ExecutionPolicy Bypass -File scripts\setup-ollama.ps1   # tuỳ chọn: Qwen local cho Summary/Analyze
python -m ai_record                                            # hoặc double-click shortcut trên Desktop

Dữ liệu của bạn ở lại với bạn

Khác với các dịch vụ đám mây gửi mọi thứ lên server, AI Record xử lý ngay trên phần cứng của bạn. So sánh nhanh cho dễ hình dung.

AI Record (cục bộ) Dịch vụ đám mây điển hình
Âm thanh cuộc họp Ở lại trên máy bạn Tải lên server nhà cung cấp
Chi phí sử dụng Miễn phí sau khi cài Trả theo phút hoặc token
Hoạt động offline Có, trừ bước tóm tắt cloud Không, luôn cần mạng
Quyền kiểm soát dữ liệu Toàn quyền, file nằm trong máy Phụ thuộc chính sách bên thứ ba

Lưu ý pháp lý: tuỳ khu vực, việc ghi âm người khác có thể cần sự đồng ý của họ. Công cụ này sinh ra để hỗ trợ ghi chú, hãy thông báo và xin phép khi luật yêu cầu.
